# Fakebook – Object Oriented Javascript Project

This is my Fakebook assignment project, created using HTML, CSS, and JavaScript.  
It includes post creation, image uploads, and a dynamic user info modal.

## Features

### ✔ Post System
- Create text posts
- Upload and preview images
- Automatic date & time on every post
- Newest posts shown at the top

### ✔ User Modal
- Opens when clicking the profile avatar
- Shows:
  - Name
  - Username
  - Email
  - Pages
  - Groups
  - Monetization status
- All fields are dynamically filled using JavaScript

### ✔ UI / Layout
- Clean modern layout  
- Dark theme  
- Fully responsive  
- Smooth modal animation
